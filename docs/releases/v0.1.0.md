# AI Coding Skills v0.1.0

First public release of `AI Coding Skills`, a curated repository of reusable AI coding skills for Claude Code and Codex.

## Highlights

- `roundtable-design-review`
  Structured multi-model design review that helps teams tighten the first proposal before risky implementation starts.
- `token-guard`
  Escalation-only token guardrail for expensive sessions, repo-wide scans, heavy tool loops, and context blowups.
- Bilingual landing pages
  Chinese is now the default repository README, with a dedicated English README alongside it.
- Launch assets
  Added GitHub metadata guidance, launch copy, release notes, and a social preview asset for public rollout.

## Why this release matters

- Stronger prompts instead of weak first drafts heading straight into code.
- Safer design review before high-risk refactors and architecture changes.
- Token-aware execution that catches waste before the session collapses.
- Portable packaging with `SKILL.md`, optional `agents/openai.yaml`, checked-in examples, and setup docs.
